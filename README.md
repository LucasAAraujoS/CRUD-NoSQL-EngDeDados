# CRUD-NoSQL-EngDeDados
Este repositório trata de um CRUD operado em um Banco de Dados NoSQL. Desenvolvido para a matéria de Engenharia de Dados sob a orientação do docente André Britto de Carvalho utilizamos uma nuvem da AWS para hospedar nossa base de dados, modelando os dados diretamente da database relacional da parte 1 e realizando operações de CRUD em tempo real.
